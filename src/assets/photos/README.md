# 📁 Drop Your Photos Here

Organize your images into subfolders:

```
photos/
├── portraits/   ← Portrait photography
├── nature/      ← Landscapes, flora, fauna
└── street/      ← Urban, documentary, street
```

Then import them in `src/data/images.js`:

```js
import shot1 from '../assets/photos/portraits/my-portrait.jpg'
```

Supported formats: .jpg · .jpeg · .webp · .png
